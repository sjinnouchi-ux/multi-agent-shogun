# Skill intake evidence

- Request phrase and scope
- Canonical repository, tag, 40-character commit, path, and content hash
- License and notice location
- Executable/network/credential/destructive-behavior review
- Codex App installation status and immutable pin
- Shogun disposition: adapted, codex-only, excluded, or pending
- User approval for a codex-only or excluded proposal
- Eligible Shogun roles and explicit non-goals
- RED/GREEN pressure evidence and exact verification commands
- Registry version, deterministic lock, review, and merged commit
- Separate post-merge Codex App and Shogun deployment results

Do not include secret values, authentication output, tmux pane content, raw queue/report bodies, or raw logs.
